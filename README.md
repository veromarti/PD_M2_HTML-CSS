Clipboard Landing Page
This project is a responsive landing page built using HTML and CSS, based on the Clipboard Landing Page challenge from Frontend Mentor.

## Project Purpose
The goal of this project is to replicate a professional landing page design while applying semantic HTML, modern CSS layout techniques, and responsive design principles.

## Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
Media Queries
Google Fonts
Responsive Design
The layout adapts correctly to:

Desktop screens (≥ 1024px)
Tablets (≥ 768px)
